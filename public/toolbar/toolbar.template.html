Liar's Dice

Instructions

How To Play

Start a game

Enter number of players and click start game

Make a move

Make a claim or challenge the last claim

Claim

Make a bid on total number of X dice with Y face between all players

The number of dice claimed must be greater than the previous claimed number, or can be equal to the previous claimed number if the claimed face is greater

Choose any number of dice with the same face to put onto the board and reroll the remaining

Challenge

If you don't believe the previous claim is true, you may challenge it and the dice will be counted
